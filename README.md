# MyWallet

React + Vite 기반의 개인 가계부 웹앱입니다.

업로드한 엑셀 가계부의 구조를 참고해 월간 요약, 거래 입력, 거래 장부, 자산 관리 화면을 먼저 잡았습니다.

## 현재 구성

- 월별 지출, 수입, 자산, 잔액 요약
- 지출과 수입을 좌우로 나눠 입력하는 거래 폼
- 지출표와 수입표를 분리한 거래 장부
- 카테고리별 지출, 수입, 자산 합계
- 브라우저 localStorage 기반 임시 저장

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버 기본 주소는 `http://localhost:5173`입니다.

## 빌드 방법

```bash
npm run build
npm run preview
```

배포 시 빌드 명령어는 `npm run build`, 결과 폴더는 `dist`로 두면 됩니다.
