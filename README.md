# MyWallet

React + Vite 기반의 개인 가계부 웹앱입니다.

## 현재 구성

- 월간 수입 / 지출 / 잔액 요약
- 최근 거래내역 샘플 리스트
- 카테고리별 지출 요약
- 예산 사용률 카드
- 추후 로컬 저장소, 로그인, 통계, 반복 지출 기능 확장 예정

## 실행 방법

```bash
npm install
npm run dev
```

Vite 개발 서버 기본 주소는 `http://localhost:5173`입니다.

## 개발 메모

현재는 실제 저장 기능 없이 mock 데이터를 기반으로 화면 구조만 잡아둔 상태입니다.
다음 단계에서 거래 추가/수정/삭제, 월별 필터, 카테고리 관리, 로컬 저장 기능을 붙이면 됩니다.
